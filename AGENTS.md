# Workspace AI Collaboration Guide

This repository builds a SvelteKit app (Svelte 5, Tailwind CSS v4, Valibot, Cloudflare Workers via Alchemy) using a Behavior-Driven Development (BDD) pipeline.

## Quick Reference

- **Package manager**: `pnpm`
- **Commands**: `pnpm dev`, `pnpm format`, `pnpm check && pnpm lint`
